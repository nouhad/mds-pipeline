# 3D2G04A_3DScripting: Day 04

## Tasks

### Task 01: Simple UI
Subclass `BaseWindow`, add a button labelled "Hello from UI", and connect it to a method that prints a message. Commit and push your change.

### Task 02: UI Wraps Tool
Build a UI with prefix and descriptor fields and a "Rename Selected" button that calls `build_name` on each selected object and updates a label showing how many were renamed. Commit and push your change.

## Submission

1. Create your branch: `git checkout -b day04/your-name`
2. Complete each task and commit after each one.
3. Push: `git push -u origin day04/your-name`
4. Open a PR titled `Day 04 - Your Name`.
