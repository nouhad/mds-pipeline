# 3D2G04A_3DScripting: Day 02

## Tasks

### Task 01: Batch Rename
Select objects in your Maya scene. Use a loop to rename each object using `mds.naming.build_name` and print a before/after report. Commit and push your change.

### Task 02: Grid Layout
Use nested loops to create a grid of spheres, space them apart, and name each one with `mds.naming.build_name`. Commit and push your change.

## Submission

1. Create your branch: `git checkout -b day02/your-name`
2. Complete each task and commit after each one.
3. Push: `git push -u origin day02/your-name`
4. Open a PR titled `Day 02 - Your Name`.
