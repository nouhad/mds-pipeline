# 3D2G04A_3DScripting: Day 03

## Tasks

### Task 01: Paths and Versions
Use `mds.paths`, `mds.versioning`, and `mds.io` to build a shot path, determine the next version, and write and read a JSON metadata file. Commit and push your change.

### Task 02: Import Folder OBJs
Find all `.obj` files in a folder, import each one into Maya using `cmds.file()`, and rename each imported object with `mds.naming.build_name`. Commit and push your change.

## Submission

1. Create your branch: `git checkout -b day03/your-name`
2. Complete each task and commit after each one.
3. Push: `git push -u origin day03/your-name`
4. Open a PR titled `Day 03 - Your Name`.
